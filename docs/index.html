AddNewProject

Function Adds new project and membership of current user to database.
Location App ( MyProjects ( AddNewProject ) )
Inputs input (new project name)
button (add new project)

BottomNav

Function Create bottom navigation.
Location App ( BottomNav )

EditableField

Function Multipurpose component for editing and saving text-based information to database.
Location App ( MyProfile ( MyDetails ( EditableField ) ) )
App ( MyProjects ( Project ( EditableField ) ) )
Inputs input (textarea) or input (text/email)
button (Enable editing/Save changes)

EditSkills

Function Display user's skills in talents-table. Add skill to user via talents-table. Find if skill exists, else create new skill to skills-table.
Location App ( MyProfile ( MyDetails ( EditSkills ) ) )
Inputs input (new project name)
button (add new project)

FetchUsers

Function Admin tool to display all users and option to delete accounts.
Location App ( MyProfile ( MyDetails ( FetchUsers ) ) )
Inputs button (delete user account)

Login

Function Login or sign up with Google.
Location App ( Signup ( Login ) )

MyDetails

Function Wrapper component for user account profile overview.
Location App ( MyProfile ( MyDetails ) )

MyProfile

Function Wrapper component to display user account information.
Location App ( MyProfile )

MyProjects

Function Display list or user's projects.
Location App ( MyProjects )

NotFound

Function Display link to return main page.
Location App ( NotFound )

Project

Function Display details and participants of single project.
Location App ( MyProjects ( Project ) )
Inputs button (delete project)

ProjectMembers

Function Display details and participants of single project.
Location App ( MyProjects ( Project ( ProjectMembers ) ) )
Inputs button (leave project)
button (remove project member)
input (search for user)
button (add new project member)

Search

Function Search for Projects, Users and Skills. Display findings in table.
Location App ( Search )
Inputs input (search keyword)
button (activate search)

Settings

Function Edit username, password or delete account.
Location App ( Settings )
Inputs input (text, new username)
button (change username)
input (password, new password)
input (password, repeat new password)
button (change password)
button (delete account)

Signup

Function Display login/signup form. Check if user is authenticated with Google login/sign up.
Location App ( Signup )
Inputs Login:
input (text, username)
input (password, )
button (login)
button (show signup form)

Signup:
input (text, new username)
input (password, new password)
input (password, repeat new password)
button (signup)
button (show login form)

SvgIcon

Function Display software as icon in user skills list
Location App ( MyProfile ( MyDetails ( EditSkills > SkillsList ( SvgIcon ) ) ) )

TopNav

Function Display current path and Sin-logo (not logged in) or user avatar (if logged in).
Location App ( TopNav )

http-hook

Function Component to send http-requests.

validators

Function Library of input validators.
